# Assistente de Segurança da Informação (ASI)

## Status do projeto
- Em desenvolvimento

## Tecnologias utilizadas
- Java
- Swing
- MySQL

## Tecnologias complementares
- Bcrypt
- AES
- PBKDF2

## Time de desenvolvedores
- Giovanny Oliveira

## Objetivo do software
Fornecer ferramentas básicas para segurança digital de indivíduos,
 sendo voltado principalmente para usuários com baixo conhecimento digital.

## Funcionalidades do sistema
- Gerenciador de senhas
- Analisador de links
- Analisador de arquivos
